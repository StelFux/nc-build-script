# Nextcloud Build Script

**This script and repo is unaffiliated with Nextcloud GmbH**

This is a build script for use with Nextcloud.

This script does not include signing or actually releasing a build.

### How to Run

```bash
npm ci
npm start -- --help
npm start -- build
```
